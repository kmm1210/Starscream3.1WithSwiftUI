#### Node.js WebSocket 서버 예제
1. Node.js가 설치되어 있는지 확인합니다. 설치되지 않았다면, Node.js 공식 웹사이트에서 설치할 수 있습니다. 

    <https://nodejs.org/en>
2. 새로운 디렉터리를 만들고, 그 안에서 `npm init -y` 명령어를 사용하여 Node.js 프로젝트를 초기화합니다.
3. ws 라이브러리를 설치합니다:

    `npm install ws`
4. WebSocket 서버 코드를 작성합니다. (server.js파일 코드 내용)
5. 서버를 실행합니다.

    `node server.js`

6. 서버 종료 control + c
